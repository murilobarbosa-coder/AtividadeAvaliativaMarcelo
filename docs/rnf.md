### Requisitos Não Funcionais (RNF)

- **RNF-01 – Desempenho**  
  O sistema deve carregar a lista de leads em até 2 segundos.

- **RNF-02 – Usabilidade**  
  A interface deve ser simples, intuitiva e responsiva para dispositivos móveis.

- **RNF-03 – Segurança**  
  O sistema deve exigir autenticação (login e senha) para acesso aos dados.

- **RNF-04 – Manutenibilidade**  
  O código do sistema deve ser organizado em camadas (models, views, viewmodels), facilitando a manutenção e evolução do projeto.

- **RNF-05 – Compatibilidade** 
  O aplicativo deve funcionar corretamente em diferentes dispositivos Android, adaptando-se a diferentes tamanhos de tela.
